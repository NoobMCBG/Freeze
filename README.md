## Commands

| **Command** | **Description** |
| --- | --- |
| **/freeze** | **Freeze Players** |
| **/unfreeze** | **UnFreeze Players** |
  
<br>
 
## Permissions

- permission `freeze.command` to use commands /freeze
- permission `unfree.command` to use commands /unfreeze
